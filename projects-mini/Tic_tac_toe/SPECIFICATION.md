# Specification: Tic_tac_toe

## Expected Functionality
This project implements Tic Tac Toe.

A python based 2-player Tic Tac Toe game.
It takes input of the two players.
The two players are named as X and O

## Input
- User inputs as required by the application
- Configuration parameters as needed

## Expected Output
- Results based on the application's functionality
- Status messages and feedback to the user

## Usage
```bash
python app.py
```
