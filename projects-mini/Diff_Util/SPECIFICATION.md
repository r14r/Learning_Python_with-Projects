# Specification: Diff_Util

## Expected Functionality
This project implements Diff Util.

This program is a minimal clone of the UNIX ``diff`` program.
``diff.py`` takes two file names as command-line arguments and compares them for changes.
* Rich: ``rich==10.11.0``

## Input
- User inputs as required by the application
- Configuration parameters as needed

## Expected Output
- Results based on the application's functionality
- Status messages and feedback to the user

## Usage
```bash
python app.py
```
