# Specification: cli_proxy_tester

## Expected Functionality
This project implements Cli Proxy Tester.

*Author:* Ingo Kleiber (ingo@kleiber.me)
This mini project is a proxy tester based on `requests`. It utilized `pandas` for handling csv files and
`click` for the CLI.

## Input
- User inputs as required by the application
- Configuration parameters as needed

## Expected Output
- Results based on the application's functionality
- Status messages and feedback to the user

## Usage
```bash
python app.py
```
