# Specification: Website_blocker

## Expected Functionality
This project implements Website Blocker.

This script lets you block websites on your computer by editing your hosts file.
First add your Blocked Websites to the array in both scripts.
On Linux: `sudo python website_blocker.py`

## Input
- User inputs as required by the application
- Configuration parameters as needed

## Expected Output
- Results based on the application's functionality
- Status messages and feedback to the user

## Usage
```bash
python app.py
```
