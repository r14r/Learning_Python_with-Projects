# Specification: RockPaperScissors_Game

## Expected Functionality
This project implements Rockpaperscissors Game.

* You can enter the number of games you want to play.
* There is also a score window which is displayed after every turn.

- Configuration parameters as needed

## Expected Output
- Results based on the application's functionality
- Status messages and feedback to the user
