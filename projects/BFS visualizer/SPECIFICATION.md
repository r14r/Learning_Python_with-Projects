# Specification: BFS visualizer

## Expected Functionality
This project implements Bfs Visualizer.

This project helps us to visualize BFS algorithm by searching end point from start in a maze.
Python
* [curses](docs.python.org/3/howto/curses.html) library to display result in terminal

## Input
- User inputs as required by the application
- Configuration parameters as needed

## Expected Output
- Results based on the application's functionality
- Status messages and feedback to the user
